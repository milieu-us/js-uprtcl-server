# Webserver Service Provider for [js-uprtcl](https://github.com/uprtcl/js-uprtcl)

To run it, install [dgraph](https://github.com/dgraph-io/dgraph) then

**Prerequisites**

`serverless` should be installed as global npm module

`npm i -g serverless`

**Local Development**

```
npm i
./run-dgraph.sh
npm run dev
```
